# JSistem

Este proyecto nace con mi necesidad personal de llevar un control estricto y preciso de todo lo que ocurre en mi negocio **JSistem** nos permite llevar un control de ventas diarias, ingresos, egresos, costos operacionales, y ganancia neta.


# Modulos

**JSistem** cuenta con gran variedad de modulos para llevar a cabo sus diferentes funciones.


## Ventas

En este modulo tendremos el registro de clientes, registro de ordenes de servicio, registro de notas de entrega(remisi√≥n), registro de facturas y registro de ventas diarias.

## Inventario

En este modulo tendremos el registro de todos los productos del establecimiento con todos sus datos tales como: fecha de ingreso, precio de costo, precio de venta, y proveedor.

## Administrativo

En este modulo podremos llevar un control total de ventas diarias, gastos diarios, estado de cuentas por pagar y cuentas por cobrar.
